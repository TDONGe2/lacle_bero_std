## 진행 방식

1. 발표자는 해당책 챕터 발표자료를 준비한다
2. 나머지 인원은 해당 챕터를 개인적으로 정리하고 발표자에게 할 질문을 준비한다.
3. 발표진행을 하며 준비된 질문을 주고 받는다.
4. 서기는 발표를 녹음한다.
5. 발표자료 및 준비한 자료는 해당 week 래포지토리에 올린다.

## 발표진행 순서
